# 💰 FinTrack

Modern Personal Finance Dashboard built with React + Vite.

## 🚀 Features

- Add Transactions
- Edit Transactions
- Delete Transactions
- Category Filtering
- Search Transactions
- Dark / Light Theme
- Expense Analytics
- Pie Chart Visualization
- Local Storage Persistence
- Responsive Design
- Smooth Animations

---

## 🛠️ Tech Stack

- React
- Vite
- CSS3
- Recharts
- Framer Motion
- Lucide React
- React Toastify
